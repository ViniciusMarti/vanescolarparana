Nossa Missão: Tecnologia para sua Tranquilidade

Utilizamos Inteligência Artificial e dados públicos para conectar pais e transportadores escolares com segurança e praticidade.

Como o Site Funciona

1. Coleta de Dados

As informações dos transportadores são coletadas de bancos de dados públicos, como os da URBS e prefeituras.

2. Organização com IA

Utilizamos a Inteligência Artificial do Gemini para organizar os dados, criar os textos e estruturar as páginas de forma clara.

3. Busca Facilitada

Você navega por bairro e encontra uma lista organizada, com contato direto para os motoristas via WhatsApp.

Sobre o Projeto

O Van Escolar Paraná é um projeto independente, idealizado e desenvolvido por mim, Vinicius Oliveira. Este site foi construído com um objetivo claro: demonstrar como a Inteligência Artificial, especificamente o modelo Gemini do Google, pode ser usada para criar ferramentas práticas e úteis para o dia a dia das famílias.

Todo o trabalho foi feito sem nenhum tipo de remuneração, com o propósito de mostrar que podemos usar IA para criar soluções que ajudem a comunidade. O projeto já alcançou a marca de 200 impressões por mês, e nossa meta é chegar a todos os pais e responsáveis do Paraná que buscam opções seguras e confiáveis de transporte para seus filhos.
